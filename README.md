# WorldCup2026 Unofficial Japan Funsite

ワールドカップ2026の非公式日本ファンサイトです。

> [!IMPORTANT]
> 個人的な感想を書いたサイトであるため誤りが含まれる場合があります。
> また各著作物や商標等は権利者に帰属します。

## 開発

Nodejs v24, PNPM v11以降が必要です。

1. 以下のコマンドで、依存関係をインストールしてください

```bash
pnpm install
```

2. 以下のコマンドで、開発サーバーを起動してください

```bash
pnpm dev
```

## ビルド

```bash
pnpm build
```
